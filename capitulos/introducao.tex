\chapter{Introdução.}

Atualmente, os sistemas web fazem parte do dia a dia das pessoas. Estas, estão cada vez mais conectadas a um mundo virtual, também denominado como \textit{``internet''}, fundamentalmente sustentado pela comunicação entre seus usuários. Para manter a interação entre eles, existem vários tipos de aplicações web, como exemplo as redes sociais, manipuladas diariamente por grande parte dos usuários interativos da internet, também conhecidos como internautas. 

Existem diversos outros tipos de aplicações para a internet, \citeonline{gonccalves2005proposta} cita como exemplares as páginas para a Web, aplicações \textit{E-business} e aplicações de comércio eletrônico (\textit{E-commerce}).
